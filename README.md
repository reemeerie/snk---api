***API para proyecto E-commerce de Zapatillas***

Este proyecto corresponde a mi primer trabajo desarrollado con Node.js. Consiste en una API REST para el e-commerce de zapatillas The Sneaker Store, enfocada en la aplicación práctica de los conceptos fundamentales del ecosistema Node.

**Tecnologías utilizadas**

- Node.js 16.x
- Bcrypt para la encriptación de contraseñas de los usuarios
- CORS para permitir la comunicación en entornos de desarrollo local
- Dotenv para la gestión de variables de entorno
- Express para la creación y manejo de endpoints
- JSON Web Token (JWT) para la gestión de autenticación y autorización de usuarios
- Mongoose como ORM para la comunicación con la base de datos MongoDB, alojada en MongoDB Atlas

**Objetivo y aprendizaje**

Este proyecto representó un primer acercamiento práctico a:

- La estructura de archivos típica de proyectos backend en Node.js
- La separación de responsabilidades entre rutas, controllers y services
- El uso de middlewares para manejar autenticación y autorización
- La comunicación con una base de datos NoSQL mediante un ORM
- La utilización de bcrypt para la seguridad de credenciales de usuarios
- La integración de JSON Web Token para la creación de usuarios, gestión de permisos y realización de órdenes de compra
